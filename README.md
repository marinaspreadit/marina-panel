# Marina Panel

Panel de gestió (MVP) per a Spreadit/Genius.

## Objectiu
- Gestionar *jobs* (p. ex. “Scrape leads”) i artefactes (CSV) amb login.
- Orquestració segura via email-jobs (Vercel → marinaspreadit@gmail.com → OpenClaw).

## Dev
```bash
npm install
npm run dev
```
